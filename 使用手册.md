## 安装与启动

### 使用 .exe 文件（推荐，无需安装任何软件）

1. 将 `ExamClient.exe` 复制到你电脑上任意位置（桌面、D 盘均可）
2. **双击** `ExamClient.exe` 启动程序
3. 首次运行时 Windows 可能弹出安全提示：

```
┌─────────────────────────────────────────┐
│  Windows 已保护你的电脑                  │
│  Microsoft Defender SmartScreen...      │
│                                         │
│  [不运行]    [更多信息]                  │
└─────────────────────────────────────────┘
```

点击「**更多信息**」，再点击「**仍要运行**」即可正常启动。

> 程序完全离线运行，不联网，不修改系统设置，数据只保存在你自己的电脑上。

---

### 如果双击没有反应

右键点击 `ExamClient.exe`，选择「**以管理员身份运行**」。

如果仍然无效，可能是杀毒软件拦截了程序。将程序文件添加到杀毒软件的「信任列表」或「白名单」中，再重新运行。

## 导入题库

使用程序前，必须先导入至少一个题库。

### 操作步骤

**第一步：** 点击左侧「**↑ 导 入**」

**第二步：** 点击「**浏览文件...**」按钮

```
┌──────────────────────────────────────────┐
│  选择题库文件                             │
│  [浏览文件...]   未选择文件               │
│                                          │
│  题库名称：[________________]            │
│                                          │
│  [开始导入]                              │
└──────────────────────────────────────────┘
```

**第三步：** 在弹出的文件选择窗口中，找到你的题库文件（`.xlsx` 或 `.docx`），双击选中

**第四步：** 「题库名称」输入框会自动填入文件名，你也可以修改为更易识别的名称（如「2024年真题」）

**第五步：** 点击「**开始导入**」，等待进度条走完

### 导入结果说明

**导入成功：**
```
✓ 解析完成，共发现 120 题
✓ 成功导入 120 题，失败 0 题
```

**部分题目跳过（不影响其他题目）：**
```
✓ 成功导入 118 题，失败 2 题
✗ 第 45 题跳过：答案为空
✗ 第 67 题跳过：题干为空
```

跳过的题目说明该题格式有问题，其余题目已正常导入，可以正常使用。

**导入完成后，** 右侧「已有题库」列表中会出现刚导入的题库，显示题库名称、题目数量和导入时间。

## 制作题库文件

### Excel 格式（推荐）

用 Excel 或 WPS 表格制作，**第一行是表头，从第二行开始填题目**。

**列的顺序固定如下（不能调换）：**

| A列 | B列 | C列 | D列 | E列 | F列 | G列 | H列 | I列 | J列 | K列 |
|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| 题型 | 题干 | 选项A | 选项B | 选项C | 选项D | 选项E | 正确答案 | 解析 | 分类 | 难度 |

**A列（题型）怎么填：**

| 你想要的题型 | A列填写 |
|------------|---------|
| 单选题 | `single` 或 `单选题` |
| 多选题 | `multi` 或 `多选题` |
| 判断题 | `truefalse` 或 `判断题` |
| 场景题 | `scenario` 或 `场景题` |

**H列（正确答案）怎么填：**

| 题型 | 填写示例 | 说明 |
|------|---------|------|
| 单选题 | `B` | 填正确选项的字母（大写） |
| 多选题 | `ABD` | 把所有正确选项的字母连在一起写 |
| 判断题 | `True` 或 `False` | True=正确，False=错误 |

**填写示例（Excel 表格内容）：**

| A | B | C | D | E | F | G | H | I | J | K |
|---|---|---|---|---|---|---|---|---|---|---|
| 题型 | 题干 | 选项A | 选项B | 选项C | 选项D | 选项E | 正确答案 | 解析 | 分类 | 难度 |
| single | 中国的首都是？ | 上海 | 北京 | 广州 | 深圳 | | B | 北京是中华人民共和国的首都 | 地理 | 1 |
| multi | 以下属于直辖市的有？ | 北京 | 上海 | 广州 | 重庆 | 天津 | ABDE | 广州是省会城市 | 地理 | 2 |
| truefalse | 长江是中国最长的河流。 | | | | | | True | 长江全长约6300公里 | 地理 | 1 |

**小技巧：** 可以用多个 Sheet（工作表）来区分不同章节，Sheet 的名称会自动作为题目的分类标签。

---

### Word 格式

用 Word 或 WPS 文字制作，按照以下固定格式书写，**题目之间必须用空行隔开**：

```
【单选题】
1. 中国的首都是？
A. 上海
B. 北京
C. 广州
D. 深圳
答案：B
解析：北京是中华人民共和国的首都。
分类：地理
难度：1

【多选题】
2. 以下属于直辖市的有？
A. 北京
B. 上海
C. 广州
D. 重庆
E. 天津
答案：ABDE
解析：广州是省会城市，不是直辖市。

【判断题】
3. 长江是中国最长的河流。
答案：True
解析：长江全长约6300公里，是中国第一长河。
```

**格式要点（必须遵守）：**
- 题型标记必须用【】括起来，如 `【单选题】`
- 选项必须以字母加点开头，如 `A.` 或 `A、`
- 答案行必须以 `答案：` 开头（冒号可以是中文：或英文:）
- 解析、分类、难度行可以省略不写
- 每道题之间必须有空行

## 更新题库

当题库有新版本时，直接重新导入即可，旧题库不会被自动删除。

**操作步骤：**

1. 点击「**↑ 导 入**」
2. 选择新版本的题库文件
3. 填写题库名称（建议加上版本号，如「2024年真题 v2」）
4. 点击「开始导入」
5. 导入完成后，右侧「已有题库」列表中会同时显示新旧两个版本
6. 如需删除旧版本：在列表中点击选中旧题库，点击「**删除选中题库**」

> 删除题库会同时删除该题库下的所有题目，但不会影响已有的考试记录和错题本。

## 数据备份与恢复

程序的所有数据（题库、考试记录、错题本）保存在你的电脑上：

**数据文件位置：**
```
C:\Users\你的用户名\AppData\Roaming\ExamClient\exam.db
```

> 如何找到这个文件夹：按 `Win + R`，输入 `%APPDATA%\ExamClient`，回车即可打开。

**备份方法：**
将 `exam.db` 文件复制到 U 盘或其他位置保存即可。

**恢复方法：**
将备份的 `exam.db` 文件复制回原来的位置，覆盖现有文件即可。

**在新电脑上使用原有数据：**
1. 在新电脑上运行一次 `ExamClient.exe`（会自动创建数据目录）
2. 关闭程序
3. 将备份的 `exam.db` 复制到新电脑的 `%APPDATA%\ExamClient\` 目录下，覆盖原文件
4. 重新启动程序，原有数据全部恢复

## 常见问题

**Q：双击 .exe 没有反应，或者一闪而过？**

A：右键点击 .exe 文件，选择「以管理员身份运行」。如果仍然无效，检查杀毒软件是否拦截了程序，将程序添加到白名单即可。

---

**Q：导入后提示「成功导入 0 题」？**

A：检查题库文件格式是否正确：
- Excel 文件：确认第一行是表头，H列（正确答案）不为空，A列（题型）填写正确
- Word 文件：确认每道题都有「答案：」行，题型标记使用【】括起来

---

**Q：程序界面文字显示为方块或乱码？**

A：这是字体问题。尝试以下方法：
1. 右键桌面 → 显示设置 → 高级缩放设置，关闭「让应用自动修复模糊问题」，重启程序
2. 如果仍然乱码，确认系统已安装「微软雅黑」字体

---

**Q：考试时不小心关闭了窗口，成绩还在吗？**

A：关闭窗口视为放弃本次考试，成绩**不会**保存。建议使用「交卷」按钮正常结束考试。

---

**Q：可以同时使用多个题库吗？**

A：可以。程序支持导入多个题库，考试时在下拉框中选择使用哪个题库即可。

---

**Q：多选题怎么选？**

A：多选题的选项是方形复选框（□），单击可以勾选或取消，可以同时勾选多个选项。单选题是圆形单选按钮（○），只能选一个。

---

**Q：判断题怎么选？**

A：判断题显示「正确」和「错误」两个选项，点击其中一个即可。

---

**Q：多选题答案顺序有影响吗？**

A：没有影响。系统判题时会忽略答案字母的顺序，选 AB 和选 BA 视为相同答案。

---

**Q：错题本里的题目太多，怎么清理？**

A：在错题本页面，对已经掌握的题目点击「标记已掌握」，这些题目会从待复习列表中隐藏。如果想彻底删除，选中后点击「从错题本删除」。

    ▼

    ├──→ 错题强化
    │       点击「错题本」→ 查看错题 → 开始错题练习

    └──→ 查看进步
